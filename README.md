Crafteholic-E-commerce Platform for Crafts
A specialized e-commerce platform for handmade crafts and unique artistic products.

Tech Stack:
1. Backend: PHP & MySQL
2.  Frontend: HTML5, CSS3, JavaScript
Setup:
1. Move files to `htdocs`.
2. Import `crafteholic_db.sql` into phpMyAdmin.
3. Access via `localhost/crafteholic`.
